###  last - виведення останніх n рядків файлу

**last** - програма, яка виводить останні n рядків із файлу або стандартного вводу.  
Програма реалізована з використанням кільцевого буфера та порядкового читання fgets, що забезпечує ефективне використання пам’яті навіть для великих файлів.

#### Можливості

- Виведення останніх n рядків
- Робота з файлами або стандартним вводом (stdin)
- Ефективне використання пам’яті
- Підтримка зворотного порядку виведення

#### Компіляція та запуск

```bash
make
./last
```

Приклад використання:

```bash
./last /etc/ca-certificates.conf
```

або

```bash
cat /etc/ca-certificates.conf | ./last -n 5 -r
```
